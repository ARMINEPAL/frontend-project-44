Игры разума

Описание проекта

Игры разума — это набор простых консольных игр.
Пользователю нужно отвечать на вопросы в консоли.
Если все ответы верны — игра считается пройденной.

Игры

В проект входят следующие игры:
 brain-even — определить, чётное ли число
 brain-calc — решить математический пример
 brain-gcd — найти наибольший общий делитель
 brain-progression — найти пропущенное число
 brain-prime — определить, простое ли число

Требования:

Для работы проекта необходимо: Node.js; npm

Установка:

1) Склонируйте репозиторий

git clone URL-репозитория

2) Перейдите в папку проекта 

cd frontend-project-44

3) Установите зависимости

npm ci

4) Установите проект 

npm link
